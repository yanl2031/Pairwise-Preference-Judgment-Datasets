Collected by Ngai Meng Kou in 2017.

-1- DATA CONTENT

Photo.csv contains pairwise comparisons between 200 photos, collected from a real crowdsourcing system, Figure Eight (https://www.figure-eight.com/). There are in total 19,900 rows. Each row is the list of user preference values between two photos.
Specifically, each row starts with three integers, two photo IDs and an integer, n, indicating the total number of user preference values received for this paired comparison. These three integers is then followed by a list of n real numbers within the range [-1, 1].
