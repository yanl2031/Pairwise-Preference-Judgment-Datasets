Originally provided by UC Berkeley. (See: http://eigentaste.berkeley.edu/dataset/)
Reproduced by Yan Li in 2017.

-1- DATA CONTENT

Jester.csv contains 100 rows, each representing a distinct joke. Each row contains 7200 distinct user ratings to the joke.
